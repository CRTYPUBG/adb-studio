# Security Policy

Supported release channels receive security fixes. Report vulnerabilities privately through
GitHub Security Advisories for the ADB Studio repository. Do not open a public issue before
coordinated disclosure.

Maintainers acknowledge reports within three business days, assess severity, prepare tests and a
fix in an embargoed branch, and publish an advisory with affected versions and mitigations.
Update, plugin and release verification fail closed. Secrets and personal device data must never be
included in telemetry or build artifacts.
