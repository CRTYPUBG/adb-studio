# Release Checklist

- [ ] `version.crty`, tag, CMake, executable and installer versions agree.
- [ ] Clean configure, warnings-as-errors build and complete test matrix pass.
- [ ] Static analysis, sanitizers, CodeQL and security review pass.
- [ ] Resource, translation, accessibility and architecture validation pass.
- [ ] Benchmarks show no unexplained regression from the previous release.
- [ ] Device and plugin compatibility matrices are current.
- [ ] SPDX and CycloneDX SBOMs, licenses, hashes and provenance are generated.
- [ ] MSIX, ZIP and Inno Setup artifacts pass clean-machine tests and signature verification.
- [ ] Changelog, release notes, audit, validation, review and rollback instructions are approved.
