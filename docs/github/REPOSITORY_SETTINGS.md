# GitHub Repository Settings

Configure `CRTYPUBG/adb-studio` with Issues, Pull Requests, Discussions, Projects, Wiki, Actions,
Dependency Graph, Dependabot alerts/security updates, code scanning, secret scanning and push
protection enabled. Enable auto-delete branches and auto-merge. Allow squash and rebase merges;
disable merge commits.

Protect `main` with pull requests, at least one approval, conversation resolution, linear history,
required signed commits, no force pushes/deletion, and required successful checks from build, test,
format, lint, CodeQL, security, architecture, resource/version and release validation workflows.
Administrators should be included unless emergency policy explicitly documents an exception.

These server-side settings require repository administrator rights. Verify them quarterly and after
GitHub plan or organization-policy changes.
