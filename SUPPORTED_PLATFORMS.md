# Supported Platforms

| Platform | Architecture | Status |
|---|---|---|
| Windows 11 | x64 | Primary, CI and release supported |
| Windows 10 | x64 | Best effort; not a release guarantee |
| Linux | x64 | Architecture-ready, product support pending |
| macOS | arm64/x64 | Architecture-ready, product support pending |

Qt 6.8.1 and MSVC 2022 are the authoritative Windows release toolchain.
