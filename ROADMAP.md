# Roadmap

1. Foundation and FluentWinUI3 shell — implemented.
2. ADB backend, USB/wireless discovery and device state machine.
3. Isolated scrcpy mirroring, recording, screenshot and audio.
4. APK, package and file management; logcat and shell tools.
5. Diagnostics, benchmarks and optimization guidance.
6. Sandboxed, signed plugin SDK and compatibility program.
7. Secure updater, MSIX, signing and cross-platform adapters.

Milestones ship only after their tests, audits, documentation and release evidence pass.
