# Wireless Setup

Wireless setup is offered only after device or platform evidence indicates availability.

1. Connect and authorize the device over USB first.
2. Open Developer options and Wireless debugging on the detected device.
3. Keep computer and device on a trusted network.
4. Pair using the on-device pairing address/code.
5. Verify the resulting network serial in ADB Studio.
6. Run latency and throughput diagnostics before mirroring or recording.

ADB Studio does not infer wireless support solely from Android version. Pairing secrets are never
written to logs or documentation.
